Memory evidence organization demo

This static demo shows what a model sees before and after the HMS Shadow evidence layer. It does not require a memory database, model endpoint, or benchmark output.

The example is illustrative. It is intended to make the architecture easy to inspect before connecting a real memory database or model endpoint.
